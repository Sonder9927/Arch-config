## i3
### information
[i3wm](https://i3wm.org)

i3 is a titling window manager.

`<Alt> + <Enter>` to open terminal.

`<Alt> + 2` to move window to window2.

`<Alt> + <Shift> + 2` to turn to window2.

`<Alr> + <Shift> + q` to close window.
